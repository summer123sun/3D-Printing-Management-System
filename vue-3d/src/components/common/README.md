# common

全项目复用的基础组件，被各业务模块大量调用，**必须先于业务模块完成**。

## 文件清单

- `PageHeader.vue` — 页头（标题 + 面包屑 + 操作区）
- `DataTable.vue` — 通用表格（封装 el-table + 分页 + 加载）
- `SearchForm.vue` — 通用搜索表单（输入/选择/日期范围）
- `FileUploader.vue` — 通用上传（支持 STL / 图片 / 文档）
- `PhotoGallery.vue` — 图片轮播（作品库用）
- `StatusTag.vue` — 状态徽章（统一色系）
- `UserAvatar.vue` — 头像 + 用户名
- `EmptyState.vue` — 空状态占位
